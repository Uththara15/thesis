# Evaluating Multilingual Sentence Embedding Models for Bilingual Course Description Similarity in Finnish Higher Education

Author: Madee Uththara Deegoda Gamage
Institution: JAMK University of Applied Sciences
Degree programme: Information and Communications Technology
Supervisor: Teemu

## Project overview
Empirical evaluation of five multilingual sentence embedding models (paraphrase-multilingual-mpnet-base-v2, stsb-xlm-r-multilingual, multilingual-e5, LaBSE, FinBERT) for semantic similarity detection between Finnish and English course descriptions sourced from JAMK's Peppi curriculum management system.

## Environment
- conda environment: thesis_env
- Python 3.10
- Windows
