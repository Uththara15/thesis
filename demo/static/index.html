Bilingual course description matching

Find the three closest Finnish courses from a course description.

Write a course description in English, Finnish, or both. The app compares it with Finnish course descriptions and returns only the top three semantic matches.

How the demo works

1
Write a course description The input can be short, long, Finnish, English, or mixed.
2
LaBSE creates embeddings The query and course descriptions are placed in the same multilingual vector space.
3
Top 3 matches are shown The results are ranked by cosine similarity score.

What the score means

The score is cosine similarity between the query embedding and each course description embedding.

How to read results

The first result is the closest semantic match. The second and third are alternative nearby courses.

Demo limitation

The interface is for retrieval and review. It does not replace final human curriculum judgement.
